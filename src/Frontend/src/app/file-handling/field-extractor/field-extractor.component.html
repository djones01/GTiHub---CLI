1 - Set Input File Delimiter

Any character may be used as a delimiter, but make sure that the chosen character is in fact the one used to delimit the fields in the input file

2 - Set Number of Row to Extract Fields from

Any number greater than 0 with 1 being the first row of data in the input file

3 - Provide Input File and Upload

Input file must be in a delimited flat file format (.txt, .dat, .csv) with the field names in the first row
